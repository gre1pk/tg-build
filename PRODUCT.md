# Product

## Register

product

## Users

Владельцы мебели в Telegram, которым нужна перетяжка (диваны, кресла, стулья). Пользуются приложением с телефона, часто присылают фото своей мебели, хотят быстро понять процесс, посмотреть ткани и связаться с мастером.

## Product Purpose

Telegram Mini App для студии перетяжки мебели. Помогает клиенту: узнать услугу, посмотреть каталог тканей, выбрать материал, оставить заявку и пообщаться с исполнителем. Успех — клиент понимает процесс, выбирает ткань и выходит на контакт с мастером без звонков и долгих объяснений.

## Brand Personality

Тёплый, надёжный, ремесленный. Спокойная уверенность мастера, а не агрессивные продажи. Визуально — текстуры, фото работ, ощущение «руками сделано».

## Anti-references

- Интернет-магазин мебели с ценниками «купить в 1 клик»
- Generic AI UI: фиолетовые градиенты, Inter + карточки в карточках
- Холодный корпоративный SaaS
- Мелкие превью 48×48 для тканей (текстура должна быть видна)

## Design Principles

1. **Фото — главный аргумент.** До/после, крупные swatch тканей, реальные работы.
2. **Путь к заявке короткий.** Главная → ткани / заявка / чат — без лишних экранов.
3. **Telegram-native.** Использовать theme params и `@telegram-apps/telegram-ui`, не ломать ощущение Mini App.
4. **Доверие через процесс.** Показать шаги: фото мебели → выбор ткани → перетяжка.
5. **Каталог тканей — ядро.** Выбор материала — ключевое действие, не побочная фича.

## Accessibility & Inclusion

- WCAG 2.1 AA для контраста текста и кнопок
- `prefers-reduced-motion` для анимаций
- Крупные зоны нажатия на мобильном
- Понятные подписи к фото тканей (название, тип, цена)

## Platform

- Telegram Mini App (React + Vite)
- Vercel: Hosting + Serverless API (auth, каталог)
- Контент (roadmap): [Supabase](https://supabase.com) — Postgres + Storage + админка `/admin`
- UI kit: `@telegram-apps/telegram-ui`
